# 🗓️ Split PPLUP — Push / Pull / Legs / Upper / Lower

→ [[SAAS]] | → [[Treino - Estrutura das Séries]] | → [[✅ Autorrevisão de Treino]]

---

## O Que É e Para Quem Serve

O PPLUP é uma divisão de **5 treinos por semana** estruturada assim:

| Dia | Treino |
|---|---|
| Segunda | Push (Empurrar) |
| Terça | Pull (Puxar) |
| Quarta | Legs (Pernas) |
| Quinta | Descanso |
| Sexta | Upper (Superior) |
| Sábado | Lower (Inferior) |
| Domingo | Descanso |

**Por que não fazer PPL×2?**
PPL repetido duas vezes na semana exige 7 dias de flexibilidade (os dias de descanso se alternam). Academias fecham no domingo. Nem todo cliente tem essa disponibilidade. O PPLUP resolve isso: estrutura fixa de 5 dias, dois descansos fixos, resultado próximo ao PPL×2.

---

## Ordem das Sessões — Regra de Prioridade

**Comece a semana pelo grupo que precisa de mais atenção.**

Sugestão padrão:
- Se o cliente quer priorizar **costas** → começar com Pull
- Se o cliente quer priorizar **pernas** → começar com Legs
- **Peito no meio da semana** → costas e pernas são pesadas na lombar; colocar Push no meio dá descanso lombar entre os dias mais exigentes

Estrutura alternativa quando pernas é prioridade:

| Dia | Treino |
|---|---|
| Segunda | Legs |
| Terça | Push |
| Quarta | Pull |
| Quinta | Descanso |
| Sexta | Upper |
| Sábado | Lower |
| Domingo | Descanso |

## Volume por Sessão — Referência

### Push (Empurrar)
Foco: Peitoral, Deltoides anterior/lateral, Tríceps

| Exercício | Séries Válidas |
|---|---|
| Supino Inclinado (compostos principal) | 2–3 |
| Supino Reto | 2 |
| Voador / Crossover | 2 |
| Elevação Lateral | 2 |
| Elevação Frontal | 2 |
| Tríceps (corda, barra ou máquina) | 2–3 |

Ajustar volume conforme prioridade: mais séries em peito, ombro ou tríceps dependendo do objetivo.

---

### Pull (Puxar)
Foco: Dorsal, Trapézio, Romboides, Deltoides posterior, Bíceps

| Exercício | Séries Válidas |
|---|---|
| Remada Curvada com Barra | 2 |
| Puxada Frontal | 2 |
| Puxada Fechada ou Remada Unilateral | 2 |
| Rosca Direta / Rosca com Halteres | 2–3 |

Volume geral: majoritariamente 2 séries válidas. 3 para bíceps se for músculo lagging.

---

### Legs (Pernas — Sessão 1)
Foco: Quadríceps, Isquiossurais, Glúteos, Panturrilhas

**Lógica recomendada:** começar pela **posterior** (terra, mesa flexora, stiff) antes dos exercícios de agachamento. Isso evita fadiga acumulada na articulação do joelho antes dos compostos pesados.

| Exercício | Séries Válidas |
|---|---|
| Stiff / Terra Romeno | 2–3 |
| Mesa Flexora | 2 |
| Agachamento Livre ou Leg Press 45° | 2–3 |
| Cadeira Extensora | 2 |
| Panturrilha em pé | 2 |

---

### Upper (Superior)
Foco: Peitoral + Costas + Ombros + Bíceps + Tríceps — **tudo em uma sessão**

**Regra de ouro:** variar os estímulos em relação ao Push e Pull dos dias anteriores. Não repetir os mesmos exercícios — usar ângulos, pegadas ou máquinas diferentes.

Estrutura sugerida:

| Grupo | Exercício | Séries Válidas |
|---|---|---|
| Peito (2 exercícios) | Ex: Supino c/ halteres + Crossover | 2 cada |
| Costas (2 exercícios) | Ex: Remada na máquina + Puxada com triângulo | 2 cada |
| Ombro (1 exercício) | Ex: Elevação Lateral | 2 |
| Bíceps (1 exercício) | Ex: Rosca Martelo | 2 |
| Tríceps (1 exercício) | Ex: Tríceps Testa ou Mergulho | 2 |

Total: **7 exercícios** — dentro da janela de 5 a 7 exercícios/sessão.

---

### Lower (Inferior — Sessão 2)
Foco: Glúteos, Isquiossurais, Quadríceps, Panturrilhas — **ângulos diferentes do dia Legs**

**Regra:** equilibrar entre posterior de coxa, quadríceps e estabilidade. Não repetir exatamente os mesmos exercícios do dia Legs.

| Exercício | Séries Válidas |
|---|---|
| Hip Thrust com barra | 2–3 |
| Leg Press 45° (se usou agachamento no Legs) | 2 |
| Cadeira Extensora parcial | 2 |
| Abdução de Quadril | 2 |
| Panturrilha sentado (sóleo) | 2 |

---

## Gestão da Lombar ao Longo da Semana

Exercícios pesados de lombar não devem se concentrar em dias consecutivos.

| Sessão | Carga Lombar |
|---|---|
| Push | Baixa — sem exercício com carga axial na coluna |
| Pull | Alta — remada curvada, terra (se presente) |
| Legs | Alta — agachamento, stiff |
| Upper | Baixa — exercícios sentados/deitados |
| Lower | Média — hip thrust, leg press (sem carga axial) |

**Critério:** Pull e Legs sempre separados por um dia de Push ou descanso.

---

## Progressão e Uso Correto

O PPLUP funciona quando o cliente foca em **evoluir dentro da estrutura**:
- Aumentar carga mantendo técnica
- Aumentar reps dentro do range definido (ex: de 6 para 8 antes de subir peso)
- Melhorar qualidade de execução

A estrutura não é o diferencial — a progressão dentro dela é.

**Pequenos ajustes geram grandes diferenças:**
- Músculo lagging? → Adicionar 1 série extra nele por sessão
- Músculo adiantado? → Manter mínimo, focar energia no atrasado
- Lombar frágil? → Substituir remada curvada por remada com apoio, terra por stiff com halteres

## Comparação com Outras Divisões

| Divisão | Frequência semanal | Dias/semana | Melhor para |
|---|---|---|---|
| Full Body | 1x cada músculo | 3 | Iniciantes |
| ABC (Push/Pull/Legs) | 1x cada músculo | 3 | Intermediários |
| ABCDE (5 dias) | 1x cada músculo | 5 | Intermediários avançados |
| PPL × 2 | 2x cada músculo | 6–7 | Avançados com horário flexível |
| **PPLUP** | **1,4x cada músculo** | **5** | **Intermediários/avançados com semana fixa** |
