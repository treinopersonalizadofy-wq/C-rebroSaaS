## Regra de Sessão

- **5 a 7 exercícios por sessão** (6 é o número ideal)
- **45 minutos ou mais** por treino
- Antes de entregar qualquer treino, verificar se todos os grupos abaixo estão cobertos ao longo das sessões da semana

## Como Usar Este Checklist

1. Monte todas as sessões da semana
2. Liste os músculos ativados em cada exercício (primários + secundários)
3. Marque os grupos acima
4. Qualquer grupo vazio → adicionar exercício ou redistribuir

### Músculos frequentemente esquecidos
- **Trapézio inferior** → Crucifixo inverso, face pull com rotação, puxada com escápula retraída
- **Deltoides posterior** → Face pull, crucifixo inverso, remada alta com abdução
- **Braquiorradial** → Rosca martelo, rosca com pegada neutra
- **Transverso abdominal** → Prancha, dead bug, Pallof press (não é treinado com abdominal supra)
- **Glúteo médio** → Abdução de quadril, step-up lateral, monster walk com elástico
- **Sóleo** → Panturrilha sentado (joelho flexionado)
- **Trapézio médio + Romboides** → Remada com cotovelo para fora, pullover, face pull

## Distribuição por Split

### 3 dias (ABC ou Full Body)
| Sessão | Grupos Primários | Grupos Secundários |
|---|---|---|
| A — Push | Peitoral, Deltoides anterior/lateral, Tríceps | Trapézio superior |
| B — Pull | Dorsal, Trapézio, Romboides, Redondo maior, Deltoides posterior, Bíceps | Braquial, Braquiorradial |
| C — Legs | Quadríceps, Isquiossurais, Glúteos, Panturrilhas | Core, Glúteo médio |

### 5 dias (PPLUP)
| Sessão | Grupos Primários |
|---|---|
| Push | Peitoral (3 porções), Deltoides anterior + lateral, Tríceps (3 cabeças) |
| Pull | Dorsal, Trapézio (médio + inferior), Romboides, Redondo maior, Deltoides posterior, Bíceps, Braquiorradial |
| Legs | Quadríceps (4 músculos), Isquiossurais (3), Glúteos (3), Panturrilhas (2) |
| Upper | Peitoral (ênfase diferente), Trapézio superior, Deltoides lateral, Bíceps ou Tríceps |
| Lower | Glúteo máximo + médio, Isquiossurais, Panturrilhas, Core completo |
