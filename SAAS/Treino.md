## Combinações Comuns

### Objetivo + Lesão
- Hipertrofia + Dor lombar → [[Treino - Hipertrofia]] + [[Lesoes - Lombar]]
- Emagrecimento + Dor no joelho → [[Treino - Emagrecimento]] + [[Lesoes - Joelho]]
- Força + Ombro → [[Treino - Força]] + [[Lesoes - Ombro]]

### Objetivo + Esporte
- Hipertrofia + Boxe → [[Treino - Hipertrofia]] + [[Esportes - Artes Marciais]]
- Condicionamento + Corrida → [[Treino - Condicionamento]] + [[Esportes - Corrida e Ciclismo]]
- Força + Futebol → [[Treino - Força]] + [[Esportes - Futebol e Coletivos]]

## Variáveis do Formulário que Determinam o Treino
- `nivel` → iniciante / intermediário / avançado
- `objetivo` → hipertrofia / emagrecimento / força / condicionamento
- `dias_semana` → 2 / 3 / 4 / 5 dias
- `tempo_treino` → 45min / 60min / 90min
- `local_treino` → academia / casa com equipamentos / casa sem equipamentos
- `esporte` → ver [[Esportes]]
- `dor_lombar` / `dor_joelho` / `dor_ombro` → ver [[Lesoes]]
