## 🎯 COMO USAR ESTE CÉREBRO

Quando chegar um email de cliente:

1. Identifica o **objetivo** → vai para o nó correspondente de Treino
2. Verifica se há **lesão** → vai para o nó de Lesões para substituições
3. Verifica se pratica **esporte** → vai para o nó de Esportes para ajuste de prioridades musculares
4. Aplica a **estrutura de séries** → [[Treino - Estrutura das Séries]]
5. Enquadra no **ciclo de periodização** → [[Treino - Periodização]]
6. Usa o **prompt de automação** → [[Automacao]]
