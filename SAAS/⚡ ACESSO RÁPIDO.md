| Objetivo | Rep Range (Válidas) | Intensidade | Séries Válidas | Descanso |
|---|---|---|---|---|
| **Hipertrofia** | **6–8 reps** | **Próximo ou falhando na 8ª** | **2–3 (máx. 3)** | 2–3 min |

## PASSO 3 — ESTRUTURA DE SÉRIES (TODO EXERCÍCIO PRINCIPAL)

```
AQUECIMENTO    →  2 × 12 reps com carga leve (metade do peso de trabalho)
FEEDER         →  1 × 6 reps com carga média (75–80% do peso de trabalho)
SÉRIES VÁLIDAS →  3 séries com carga máxima  ← NUNCA passar de 3
```

**Exemplo — Supino reto, peso de trabalho 80kg:**
- Aquecimento: 2×12 com 40kg
- Feeder: 1×6 com 62kg
- Válidas: **3×6–8 com 80kg**

> ⚠️ **Regra fixa:** séries válidas = **2 a 3 por exercício**, independente do objetivo.

### ⚠️ REGRA DO AQUECIMENTO POR GRUPO MUSCULAR
**O aquecimento (+ feeder) é feito SOMENTE no primeiro exercício de cada grupo muscular.**
Se já aqueceu peito no supino reto → supino inclinado vai direto para as séries válidas.
Se já aqueceu costas na puxada → remada vai direto para as séries válidas.

| Situação | Aquecimento? |
|---|---|
| Primeiro exercício do grupo muscular | ✅ Aquecimento + Feeder |
| Segundo+ exercício do mesmo grupo | ❌ Direto para as válidas |
| Primeiro exercício de um grupo novo | ✅ Aquecimento + Feeder |
| Exercício de prevenção / core / sport | ❌ Direto |

### ⚠️ REGRA DE REP RANGE PARA MÚSCULOS COM VOLUME BAIXO
Quando um músculo acumula poucas séries semanais (ex: bíceps e tríceps em split de 3 dias), usar **6–8 reps com carga mais pesada** em vez de 10–12.

**Lógica:** menos séries na semana = cada série precisa ter mais estímulo. Carga mais pesada (6–8 reps) garante esse estímulo. Com 10–12 reps a carga fica moderada e o estímulo por série cai.

| Músculo | Séries/semana (3 dias) | Rep range |
|---|---|---|
| Peito / Costas | 9–12 | 6–8 reps compostos · 12–15 isolados |
| Bíceps / Tríceps | 6–9 | **6–8 reps** (carga pesada) |
| Ombros | 6–9 | 6–8 compostos · 12–15 isolados |

## PASSO 4 — LESÕES → SUBSTITUIÇÕES

### 🔴 DOR LOMBAR
| TROCAR | POR |
|---|---|
| Agachamento livre | Leg Press 45° |
| Terra convencional | Terra romeno c/ halteres |
| Remada curvada c/ barra | Remada sentada no cabo |
| Desenvolvimento em pé | Desenvolvimento sentado |
| Avanço livre | Step-up no banco |
**ADICIONAR:** Prancha, Bird-dog, Hip Thrust, Dead bug

### 🔴 DOR NO JOELHO
| TROCAR | POR |
|---|---|
| Agachamento profundo | Leg press até 90° (parcial) |
| Cadeira extensora completa | TKE com elástico (últimos 30°) |
| Avanço/lunge | Step-up no banco |
| Agachamento búlgaro | Hip Thrust |
**ADICIONAR:** Abdução de quadril, TKE com elástico, Glute bridge unilateral

### 🔴 DOR NO OMBRO
| TROCAR | POR |
|---|---|
| Desenvolvimento atrás da cabeça | Desenvolvimento frontal c/ halteres |
| Puxada atrás da cabeça | Puxada frontal |
| Remada alta | Face pull |
| Supino c/ pegada muito aberta | Supino c/ halteres (pegada neutra) |
| Crucifixo amplitude total | Crossover no cabo (controlado) |
**ADICIONAR:** Face pull 3×15, Rotação externa no cabo 3×15 (OBRIGATÓRIO)

## PASSO 9 — AUTORREVISÃO OBRIGATÓRIA ANTES DE ENTREGAR

→ Ver protocolo completo em [[✅ Autorrevisão de Treino]]

Checar rapidamente antes de entregar qualquer treino:

**Rep ranges batem com o foco?**
| Foco | Séries Válidas | Intensidade obrigatória |
|---|---|---|
| **Hipertrofia** | **6–8 reps** | **Na 8ª rep não conseguiria fazer mais 1** |
| Força | 3–6 reps | Falhando na última |
| Esporte / Condicionamento | 10–15 reps | Moderado |
| Emagrecimento | 10–15 reps | Próximo da falha |
| Iniciante | 10–12 reps | Controlado |

**Descanso correto?**
| Tipo | Descanso |
|---|---|
| Composto pesado (6–8 reps) | **2,5–3 min** |
| Composto secundário (10–12 reps) | **2 min** |
| Isolado (12–15 reps) | **90s** |
| Core / funcional / sport | **60–75s** |

**7 perguntas finais:**
- [ ] Rep ranges batem com o objetivo declarado (hipertrofia vs. esporte)?
- [ ] Descanso correto por tipo de exercício?
- [ ] Lesões substituídas (nunca removidas) + fortalecimento específico incluído?
- [ ] Bloco de esporte posicionado APÓS o bloco de hipertrofia?
- [ ] Volume semanal dentro dos parâmetros científicos?
- [ ] Sessão cabe no tempo solicitado (usar fórmula: 5 + compostos×18 + isolados×7 + core×5)?
- [ ] Periodização de 12 semanas clara e explicada?

Se algum "não" → corrigir antes de entregar.
