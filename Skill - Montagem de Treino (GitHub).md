# Skill — Montagem de Treino TRAINFY

> Leia este arquivo INTEIRO antes de montar qualquer treino. Todas as regras estão aqui.

---

## FILOSOFIA DO TRAINFY

O TRAINFY usa o estilo low volume, alta qualidade:

- 2–3 séries válidas por exercício — nunca mais. Com aquecimento + feeder corretos, 2–3 séries com máxima qualidade geram mais hipertrofia do que 5 séries onde as últimas chegam com queda de performance.
- Boa carga — próximo da falha dentro do range, nunca acomodado.
- Execução limpa — forma correta sempre acima de peso maior.
- Descanso adequado — respeitado por ciência, não por pressa.
- Progressão explícita — o cliente sabe que ao bater o topo do range com boa forma, aumenta a carga.

Academia é estímulo. O crescimento acontece no descanso. Treinar mais não é treinar melhor.

## 4. ESTRUTURA DE SÉRIES — REGRA FIXA

1º EXERCÍCIO DE CADA GRUPO MUSCULAR:
- AQUECIMENTO = 2x12 com ~50% da carga de trabalho
- FEEDER (AJUSTE) = 1x6 com ~75% da carga de trabalho
- SÉRIES VÁLIDAS = 2–3 séries com carga máxima

2º, 3º EXERCÍCIO DO MESMO GRUPO:
- Vai direto para séries válidas — sem aquecimento, sem feeder.
- O músculo já está aquecido pelo exercício anterior.

Regra absoluta: nunca mais de 3 séries válidas por exercício.

### Séries válidas por objetivo:

| Objetivo | Séries válidas | Intensidade |
|---|---|---|
| Hipertrofia | 2–3 | Na última rep não conseguiria fazer mais 1 |
| Força | 2–3 | Falha concêntrica na última |
| Emagrecimento | 2–3 | RIR 2–3 |
| Condicionamento | 2–3 | RIR 3 |
| Iniciante | 2 | RIR 3–4 |

### Trabalho indireto dos compostos — REGRA CRÍTICA

- Tríceps em Push: cada supino já trabalha o tríceps indiretamente. Após 2–3 supinos, o tríceps está aquecido — não precisa de aquecimento próprio. Com 3 séries indiretas dos supinos + 1 exercício direto (2–3 séries), o tríceps atinge o MEV com 1 único exercício direto.
- Bíceps em Pull: cada puxada/remada ativa o bíceps indiretamente, mas com menos intensidade. O bíceps ainda precisa de 2 exercícios diretos para atingir o volume adequado.
- Ombro frontal no Desenvolvimento: o próprio desenvolvimento já é o exercício de ombro frontal — não há dupla cobertura.

## 5. DESCANSO ENTRE SÉRIES

Schoenfeld et al. (2016): descanso maior preserva volume por sessão → mais hipertrofia.

| Tipo de exercício | Descanso |
|---|---|
| Composto pesado (1º do grupo — supino, puxada, leg press) | 2:30–3 min |
| Composto secundário (2º do mesmo grupo) | 2 min |
| Isolador (pec deck, elevação lateral, rosca) | 90s |
| Reabilitação e core (bird dog, prancha, abdução leve) | 60–75s |

Sempre especificar o descanso em cada exercício. O descanso correto é parte do protocolo.

Estratégia: distribuir o volume em múltiplos exercícios com ângulos diferentes — nunca acumular 10 séries em um único aparelho.

## 7. PRIORIZAÇÃO DE GRUPAMENTO MUSCULAR

Quando o cliente quer focar em um grupo, adicionar séries — não exercícios.

Regra:
- Padrão: 2 séries válidas por exercício
- Com foco no grupamento: 3 séries válidas nos exercícios daquele grupo
- Máximo absoluto: 3 séries — nunca ultrapassar

---

## 8. PADRÃO DE MOVIMENTO — VARIAÇÃO OBRIGATÓRIA

O estímulo só muda quando o padrão de movimento muda — não basta mudar o ângulo.

Máximo 2 exercícios do mesmo padrão por grupo. O 3º muda o padrão.

| Grupo | Padrão A — empurrar/puxar | Padrão B — adução/isolado |
|---|---|---|
| Peitoral | Supino (inclinado, reto, declinado) | Voador, Pec Deck, Crossover |
| Costas | Puxada, Remada | Pullover |
| Ombro | Desenvolvimento | Elevação lateral, Puxada Alta para o Rosto |
| Quadríceps | Leg Press, Agachamento | Cadeira Extensora |
| Posterior | Stiff, Terra romeno (padrão do quadril) | Mesa Flexora (padrão do joelho) |

POSTERIOR DE COXA — REGRA CRÍTICA:
Stiff/Terra Romeno e Mesa Flexora são padrões DIFERENTES — usar um de cada.
Dois tipos de Mesa Flexora (deitada + sentada) = mesmo padrão duas vezes = estímulo repetido.
- CERTO: Stiff Halteres (quadril) + Mesa Flexora Deitada (joelho)
- ERRADO: Mesa Flexora Deitada + Mesa Flexora Sentada

## 9. DESCANSO ENTRE OS DIAS DE TREINO — REGRA ABSOLUTA

NUNCA 4 ou 5 dias consecutivos. Sempre uma quebra após no máximo 3 dias seguidos.

| Frequência | Distribuição correta |
|---|---|
| 2 dias (Full Body) | Treina → 1–2 dias descanso → treina. Nunca 2 dias seguidos. Ex: Seg / Qui |
| 3 dias | Seg / Ter / Qua → Qui DESCANSO |
| 4 dias | Seg / Ter → Qua DESCANSO → Qui / Sex |
| 5 dias | Seg / Ter / Qua → Qui DESCANSO obrigatório → Sex / Sáb → Dom DESCANSO |

### Sobreposição entre treinos — respeitar 48h mínimo:

| Par de treinos | Músculos compartilhados | Regra |
|---|---|---|
| Push + Upper | Ombro e tríceps | Mínimo 48h — nunca consecutivos |
| Pull + Upper | Bíceps e costas | Mínimo 48h — nunca consecutivos |
| Pull + Legs | Região lombar | NUNCA consecutivos — Push entra como buffer entre eles |
| Legs A + Legs B | Posterior e glúteo | Mínimo 48h |
| Push + Pull | Não compartilham grupos principais | Podem ser consecutivos |

## 11. DIVISÃO POR DIAS

### 2 dias → Full Body A + Full Body B

Nunca misturar peito com costas no mesmo treino.
Nunca fazer os dois dias seguidos — mínimo 1–2 dias de descanso entre eles.

| Treino | Grupos |
|---|---|
| Full Body A — Anterior | Peitoral (sup/médio/inf) + Ombro frontal + Ombro lateral + Quadríceps + Bíceps + Panturrilha em pé |
| Full Body B — Posterior | Costas (dorsal + trapézio médio + trapézio inf) + Ombro de trás + Posterior + Glúteos + Tríceps + Panturrilha sentado |

---

### 3 dias → Push / Pull / Legs

Distribuição: Seg / Ter / Qua → Qui DESCANSO

| Treino | Grupos | Exercícios |
|---|---|---|
| Push | Peitoral (3) + Ombro (2) + Tríceps (1–2) | 6–7 |
| Pull | Dorsal (2) + Trapézio médio (1) + Trapézio inf (1) + Bíceps (2) + Panturrilha sentado (1) | 6–7 |
| Legs | Quadríceps composto (1) + Quadríceps isolado (1) + Posterior (1) + Glúteo principal (1) + Glúteo lateral (1) + Panturrilha em pé (1) | 6–7 |

---

### 4 dias → Upper / Lower 2x

Distribuição: Seg (Upper 1) / Ter (Lower 1) → Qua DESCANSO → Qui (Upper 2) / Sex (Lower 2)

| Sessão | Grupos | Exercícios |
|---|---|---|
| Upper 1 | Peito (2) + Costas (2) + Ombro (1) + Bíceps (1) + Tríceps (1) | 7 |
| Lower 1 | Posterior (1) + Quad (2) + Glúteo (1) + Adutor (1) | 5 |
| Upper 2 | Peito (2) + Costas (2) + Ombro (1) + Bíceps (1) + Tríceps (1) | 7 |
| Lower 2 | Exercícios DIFERENTES do Lower 1 | 5 |

Upper 1 → AGRUPADO: peito/peito → costas/costas → ombro → bíceps → tríceps
Upper 2 → ALTERNADO: peito → costas → peito → costas → ombro → bíceps → tríceps

---

### 5 dias → PPL + UL (Push / Pull / Legs / Upper / Lower)

Distribuição fixa: Seg / Ter / Qua → Qui DESCANSO → Sex / Sáb → Dom DESCANSO

ORDEM OBRIGATÓRIA para Seg/Ter/Qua: Pull → Push → Legs
Pull na segunda, Push na terça, Legs na quarta — Push entra como buffer entre Pull e Legs (lombar).
ERRADO: Push(Seg) → Pull(Ter) → Legs(Qua) = Pull e Legs consecutivos → lombar sem recuperação.
CERTO: Pull(Seg) → Push(Ter) → Legs(Qua) = Push separa Pull de Legs.

Clientes com esporte em dia fixo (ex: Boxe na quarta com Legs):
Manter a ordem Pull → Push → Legs. O esporte na quarta coincide com o dia de Legs — correto para esportes de força superior (boxe, MMA).
Esportes de costas/bíceps (escalada, BJJ, natação) na quarta: emparelhar com Push (terça), não com Legs.

---

### 6 dias → PPL 2x (Push / Pull / Legs × 2)

REGRA CRÍTICA: Pull e Legs NUNCA em dias consecutivos. Push entra sempre como buffer.
Ordem correta: Pull → Push → Legs ou Legs → Push → Pull

Ciclo de 8 dias (3 treinos → 1 descanso → 3 treinos → 1 descanso):

| Dia | Treino |
|---|---|
| 1 | Pull |
| 2 | Push |
| 3 | Legs |
| 4 | DESCANSO OBRIGATÓRIO |
| 5 | Pull |
| 6 | Push |
| 7 | Legs |
| 8 | DESCANSO OBRIGATÓRIO |

Variação entre as duas rodadas:
- Push rodada 1: ênfase peito | Push rodada 2: ênfase ombro
- Legs rodada 1: inicia por quad | Legs rodada 2: inicia por posterior

## 12. LESÕES

### Lombar

Substituições obrigatórias:
- Terra → Stiff com halteres
- Agachamento livre → Leg Press
- Remada curvada → Remada unilateral apoiada no banco

Aquecimento APENAS nos dias com carga lombar direta (Pull, Legs) E apenas se o cliente declarou lesão lombar.
NÃO colocar em Push ou Upper. Se o cliente NÃO tem lombar: não incluir.
- Inseto Morto (Bird Dog) — 3x10 cada lado — 60–75s
- Prancha — 3x30s — 60–75s
- Elevação Pélvica leve — 3x15 — 60s

---

### Joelho

Substituições obrigatórias:
- Agachamento livre → Leg Press
- Cadeira extensora completa → Parcial 0–60°
- Afundo → Subida no Banco

Aquecimento (dias de Legs):
- Elevação Pélvica — 3x15 — 2 min
- Abdução de quadril — 3x15 — 90s
- Mesa Flexora — 3x12 — 90s

---

### Ombro

Substituições obrigatórias:
- Supino com barra → Halteres
- Desenvolvimento com barra → Halteres
- Elevação lateral com halteres pesados → Polia

Aquecimento (dias de Push e Upper):
- Rotação externa com elástico — 3x15 — 60s
- Puxada Alta para o Rosto — 3x15 — 60s

---

### Punho

Substituições obrigatórias em TODOS os exercícios — eliminar barra reta:
- Supino barra reta → Supino halteres ou máquina
- Rosca direta barra reta → Rosca EZ, Rosca halteres, Rosca Martelo ou Rosca Polia
- Desenvolvimento barra → Desenvolvimento halteres ou máquina
- Remada curvada barra → Remada máquina ou Remada halteres
- Tríceps barra reta → Tríceps Corda ou Tríceps Francês com haltere

Aquecimento antes de Push e Upper: 2 min de rotação de punho e mobilidade de antebraço.

---

### Cotovelo

Substituições obrigatórias:
- Tríceps Testa barra reta → Tríceps Corda ou Tríceps Francês haltere
- Paralela com dor → Tríceps Corda ou Extensão Overhead Polia
- Rosca direta barra reta → Rosca EZ ou Rosca halteres
- Nunca travar o cotovelo na extensão completa sob carga

Aquecimento antes de Push e Pull: rotação de cotovelo + mobilidade de antebraço — 60s cada.

---

### Cervical

Substituições obrigatórias:
- Desenvolvimento barra → Desenvolvimento máquina
- Puxada atrás da cabeça → Puxada Frontal (nunca puxar atrás do pescoço)
- Encolhimento de ombros pesado → remover completamente
- Nenhum exercício com barra ou carga sobre o pescoço

Aquecimento antes de Push e Upper: mobilidade cervical suave — rotação e inclinação lateral lenta — 10 reps cada lado.

---

### Quadril

Substituições obrigatórias:
- Agachamento livre profundo → Leg Press com amplitude limitada a 90°
- Afundo/avanço profundo → Subida no Banco
- Agachamento Sumo → Leg Press com pés levemente afastados

Aquecimento antes de Legs: Circling de quadril 10 cada lado + alongamento de quadril 30s cada lado + Caminhada com Elástico 15 passos — 60–75s entre cada.

## 13. ESPORTES — AJUSTES DE PRIORIDADE

Academia para esportistas = complemento, nunca concorrente. Max 5–6 exercícios por sessão.

| Esporte | Músculos prioritários | Ajuste principal |
|---|---|---|
| Boxe / Muay Thai / MMA / BJJ | Core, ombro, posterior de coxa | Foco em estabilidade e potência rotacional |
| Futebol / Basquete / Vôlei | Pernas, glúteo, core | Dobrar volume de pernas, reduzir upper |
| Corrida / Ciclismo | Posterior de coxa, glúteo, core | Fortalecer posterior, não sobrecarregar pernas |
| Natação | Ombro, costas, core | Usar halteres, evitar barra — proteger ombro |
| Tênis | Ombro, antebraço, core | Equilíbrio bilateral, fortalecer ombro de trás |
| Escalada / Remo / Surfe / BJJ | Costas, bíceps, core | Emparelhar dia de esporte com Push (não Pull) |

Dias COM esporte: treino leve — −1 série por exercício + RIR 1–2 a mais.
Dias SEM esporte: treino pesado — volume e intensidade completos.

EMPARCELAMENTO ESPORTE + ACADEMIA — REGRA CRÍTICA:

Prioridade 1: o dia do esporte deve coincidir com o dia de DESCANSO da academia.
Após 2–3 dias de treino já há um descanso obrigatório — encaixar o esporte nesse dia sempre que possível.

Prioridade 2 (esportes muito exigentes com espaço disponível): maximizar o descanso antes do esporte.
Para escalada, BJJ, natação, remo — quando o cliente tem dias livres — priorizar maior intervalo entre academia e esporte.
Exemplo: escalada na quarta, treino 2x/semana → academia segunda e sexta (não segunda e terça).
Isso NÃO é regra universal — só quando o esporte é muito exigente e há flexibilidade de dias.

Quando não for possível dar descanso no dia do esporte — especialmente se pratica mais de 1x/semana:
Colocar treino com intensidade reduzida:
- −1 série válida por exercício
- RIR 1–2 a mais (parar longe da falha)
- Priorizar exercícios sem sobreposição com o esporte
- Indicar no treino: "Dia de esporte — intensidade reduzida. O esporte é a prioridade."

Quando for inevitável academia + esporte no mesmo dia:
- Esportes de costas/bíceps (escalada, remo, natação, BJJ): emparelhar com PUSH — nunca com Pull
- Esportes de membros inferiores (futebol, corrida, ciclismo): emparelhar com Pull ou Push — nunca com Legs
- Esportes de força superior (boxe, MMA): emparelhar com Legs ou Lower

A regra Pull+Legs nunca consecutivos se mantém independentemente do dia de esporte.

## 13.5 ABDOMEN — REGRA

Abdômen é músculo — tem fadiga, precisa de recuperação. Não treinar todos os dias.

Frequência: 2x por semana, mínimo 48h entre as sessões.
Preferência de dia: Upper / Push.
Posicionamento: sempre no FINAL da sessão.
Exercícios de abdômen NÃO contam no limite de exercícios por sessão.

| Split | Quando colocar abdômen |
|---|---|
| 4 dias Upper/Lower | Upper 1 + Upper 2 |
| 3 dias PPL | Push + Pull (ou Push + Legs) |
| 5 dias PPLUP | Push + Upper |
| 6 dias PPL 2x | Push rodada 1 + Push rodada 2 |

Exercícios seguros para lombar sensível:
- Crunch na Polia — reto abdominal, sem compressão lombar
- Prancha Lateral — oblíquos, lombar protegida
- Inseto Morto — core profundo, excelente para lombar
- Prancha Côncava — reto abdominal, lombar plana no chão
- Elevação de Pernas Controlada — infra-abdominal, joelhos dobrados se houver dor
- Anti-Rotação na Polia — oblíquos, ideal para lombar
- Abdominal Bicicleta — reto + oblíquos

Volume: 3 séries x 15–20 reps (ou 3 x 30–45s para estáticos) | Descanso: 60–75s
Variar o exercício entre as 2 sessões — nunca repetir o mesmo nas duas.

## 15. AUTORREVISÃO — OBRIGATÓRIA ANTES DE ENTREGAR

- Distribuição semanal: nunca 4–5 dias seguidos | Push/Upper não consecutivos | Pull/Upper não consecutivos | Pull/Legs não consecutivos
- Limite de exercícios: nenhum dia ultrapassa o limite da sua categoria
- Volume: grupos grandes têm mínimo 3 exercícios no dia principal (sem esporte)
- Ordenação: compostos antes de isoladores | grupo maior antes do menor | grupos agrupados
- Padrão de movimento: max 2 do mesmo padrão por grupo | 3º exercício muda o padrão
- Estrutura de séries: 1º exercício de cada grupo tem aquec+feeder | demais vão direto
- Trabalho indireto: tríceps em Push não precisa de aquecimento | bíceps em Pull precisa de 2 exercícios diretos
- Descanso: composto pesado = 2:30–3min | secundário = 2min | isolador = 90s | reab = 60–75s
- Rep ranges: compostos 6–8 | isoladores 10–15 | leves 10–20
- Priorização: foco em grupamento = +1 série válida (2→3), não +1 exercício
- Lesões: ativar protocolo APENAS para lesões declaradas — nunca ativar aquecimento de lesão não declarada
- Lombar: aquecimento (Inseto Morto/Prancha) APENAS se cliente tem lombar declarada E APENAS em Pull/Legs
- Não repetir exercícios: mesmo exercício max 1x/semana | Legs B ≠ Legs A | Upper 2 ≠ Upper 1 | Lower ≠ Legs (incluindo isoladores como Cadeira Extensora)
- Linguagem: nenhum nome em inglês — usar sempre o nome brasileiro
- Abdômen: 2x/semana | Upper/Push preferencialmente | 2 exercícios diferentes entre as sessões
- Gênero: homens — sem exercícios com tornozeleira em posição empinada para glúteo
- Emparcelamento esporte: esportes de costas/bíceps emparelhados com Push, não Pull
- Nota final: aviso de fisgada + sono/hidratação para todos | reabilitação se lesão

## 17. REGRAS UNIVERSAIS — TODOS OS CLIENTES SEM EXCEÇÃO

Aviso de fisgada: "Se sentir fisgada ou dor aguda — pare imediatamente. Reduza a carga ou a amplitude. Se persistir, pule o exercício naquela sessão."

## 19. BANCO DE EXERCÍCIOS E VARIEDADE OBRIGATÓRIA

REGRA DE VARIEDADE — CRÍTICA:
Cada treino deve parecer diferente do anterior. Não existe exercício padrão — o banco é amplo e deve ser explorado.

Critérios de variação — usar ao menos 2 por grupo a cada treino:
- Padrão de movimento diferente (empurrar → aduzir → isolar)
- Ângulo diferente (inclinado → reto → declinado)
- Equipamento diferente (barra → halteres → máquina → polia)
- Posição diferente (bilateral → unilateral → apoiado)

Exercícios que NÃO podem aparecer sempre — rotacionar obrigatoriamente:

PEITORAL — rotacionar entre:
Supino Inclinado Halteres | Supino Inclinado Máquina | Supino Inclinado Barra | Supino Reto Halteres | Supino Reto Máquina | Supino Declinado Halteres | Paralela com tronco inclinado | Pec Deck | Crossover Polia
